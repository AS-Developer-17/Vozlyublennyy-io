# возлюбленный.IO

A richer, larger love companion app for inspiration, entertainment, and creative connection. This project includes quizzes, games, generators, and tools that help you explore love languages, create romantic messages, discover date ideas, and evaluate compatibility.

## Features

- Love language quiz with personalized results
- Romantic quote and message generator
- Compatibility calculator and relationship tools
- Date idea generator with themed suggestions
- Multiple interactive mini-apps in one experience
- Responsive UI for desktop and mobile

## Why This App

Vozlyublennyy.IO is designed to be a bigger, more engaging experience for love and relationships. It serves as a playful companion for anyone who wants to learn more about their emotional style, surprise a partner, or enjoy lighthearted dating content.

## Installation

```bash
npm install
npm run dev
```

## Technology Stack

- Vite
- React
- Firebase
- JavaScript
- CSS / responsive design

## Project Structure

- `src/components` — UI components and mini-apps
- `src/services` — services and API interactions
- `src/data` — quiz and generator content
- `public` — static assets

## How to Run

1. Clone the repository
2. Install dependencies
3. Start the local development server
4. Open `http://localhost:5173`

## Notes

The app is intended for personal use and entertainment, with a larger scope of content and interactive features.
